# Guitar Pitch UI

hosted on https://lukehowkins.github.io/guitar-pitch-ui/

Uses [vexflow](http://www.vexflow.com/build/docs) for the rendering of musical notation

## Local setup

requires node 18

`npm i`

`npm start`

 http://localhost:5173/guitar-pitch-ui/
